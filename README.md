# OS-Assignment

The assignment of the Operating System Spring 2018, implemented in C# .NET framework.

Content including:

- Banker's Algorithm
- Process Scheduling Algorithms
  - HRRN
  - Round-Robin
- Page Replacement Algorithms
  - LRU
  - FIFO
  - Pro-CLOCK
- Disk Scheduling Algorithms
  - FCFS
  - SSTF
  - SCAN
  - CSCAN
  - NStepSCAN
- Simple File System Simulator
